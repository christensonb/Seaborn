This is an example of a flask server using the Seaborn flask library.
The folder "bindings" holds the autogenerated client bindings for python and C# Unity.
The folder "endpoints" holds the rest endpoints using an MVC approach and flask blueprints.
The folder "settings" holds the configuration and a global_import folder for all endpoints to import
The folder "test" holds the unit test for the flask server.
The folder "template" holds the html template code
The folder "static" holds static files like images.
